Numerologia Cabalistica

Estou usando como base o livro "Numerologia Cabalistica" do José Carlos Rosa.

Eu já tinha criado uma base anteriormente em java, deixei na pasta Numerologia-Java. 
Esse projeto não vai ficar em java. quero aproveitar apenas a logica dos calculos.
